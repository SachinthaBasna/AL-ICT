IP Address Basics

What is an IP Address?

An IP (Internet Protocol) address is a unique 32-bit identifier assigned to each device on a network. Think of it as a postal address for your computer.

Example: 192.168.1.10
Binary: 11000000.10101000.00000001.00001010
  • Consists of 4 octets (8 bits each)
  • Each octet ranges from 0 to 255
  • Total: 32 bits = 2³² = ~4.3 billion addresses

Subnet Mask

What is a Subnet Mask?

A subnet mask separates the network portion from the host portion of an IP address.

Binary Representation:
255 = 11111111 (Network bits)
0 = 00000000 (Host bits)
Common Masks:
  • 255.0.0.0 = /8 (Class A)
  • 255.255.0.0 = /16 (Class B)
  • 255.255.255.0 = /24 (Class C)
  • 255.255.255.128 = /25
How to Find Network ID:
Perform AND operation between IP address and Subnet Mask
Example:
IP: 192.168.50.45 (11000000.10101000.00110010.00101101)
Mask: 255.255.255.0 (11111111.11111111.11111111.00000000)
Network ID: 192.168.50.0

Network ID & Host ID

Network ID

Definition: Identifies the entire network (all host bits = 0)

IP: 172.16.50.100/16
Network ID: 172.16.0.0

Cannot be assigned to devices

Host ID

Definition: Identifies a specific device within the network

IP: 192.168.10.25/24
Host ID: 25

Must be unique within network

Broadcast Address

Direct Broadcast Address

Sends data to ALL devices on a specific network (all host bits = 1)

Example 1:
Network: 192.168.10.0/24
Broadcast: 192.168.10.255
Example 2:
Network: 172.16.0.0/16
Broadcast: 172.16.255.255
Sri Lankan Example:
A university lab with network 10.10.5.0/24 has broadcast address 10.10.5.255 to reach all 254 computers simultaneously.

Subnetting

What is Subnetting?

Dividing a large network into smaller sub-networks for better management and security.

Formulas:
• Number of Subnets = 2ⁿ (n = borrowed bits)
• Hosts per Subnet = 2ʰ - 2 (h = host bits)
• Block Size = 256 - subnet mask value
Example: Subnetting 192.168.10.0/24

Need 4 subnets → Borrow 2 bits (2² = 4)

New Mask: 255.255.255.192 or /26
